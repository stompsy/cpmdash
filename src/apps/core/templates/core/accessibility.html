{# Introduction #}

The City of Port Angeles Fire Department is committed to ensuring that our CPM Metrics dashboard is accessible to all users, including those with disabilities. We strive to meet or exceed the accessibility standards outlined in the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA.

Our Commitment

We believe that everyone should be able to access and benefit from our community paramedicine data and insights. Accessibility is an ongoing effort, and we continuously work to improve the user experience for all individuals, regardless of ability or technology.

Accessibility Features

The CPM Metrics dashboard includes the following accessibility features:

Keyboard Navigation

  • Full keyboard accessibility for all interactive elements
  • Logical tab order throughout the interface
  • Visible focus indicators for keyboard users
  • Skip navigation links to bypass repetitive content

Screen Reader Support

  • Semantic HTML markup for proper content structure
  • ARIA labels and landmarks for navigation
  • Alternative text for images and icons
  • Descriptive link text and button labels
  • Proper heading hierarchy for easy navigation

Visual Design

  • High contrast color schemes that meet WCAG AA standards
  • Dark mode option to reduce eye strain
  • Resizable text that scales up to 200% without loss of functionality
  • Clear visual indicators for interactive elements
  • Consistent and predictable layout throughout the dashboard

Data Visualization Accessibility

  • Alternative data tables for all charts and graphs
  • Color-blind friendly color palettes
  • Pattern fills and textures in addition to color coding
  • Detailed descriptions of data visualizations

Standards Compliance

We are committed to conforming to the following standards:

  • WCAG 2.1 Level AA: Web Content Accessibility Guidelines published by the W3C
  • Section 508: U.S. federal accessibility standards
  • ADA Title II: Americans with Disabilities Act requirements for state and local government services

Known Limitations

While we strive for full accessibility, we acknowledge the following current limitations:

  • Some third-party embedded content may have accessibility barriers
  • Complex data visualizations may require additional context for screen reader users
  • Certain legacy features are being updated to meet current accessibility standards

We are actively working to address these limitations and welcome feedback on any accessibility barriers you encounter.

Assistive Technologies

The CPM Metrics dashboard is designed to be compatible with the following assistive technologies:

  • Screen readers (NVDA, JAWS, VoiceOver)
  • Screen magnification software
  • Speech recognition software
  • Keyboard-only navigation

Feedback and Contact Information

We welcome your feedback on the accessibility of the CPM Metrics dashboard. If you encounter accessibility barriers or have suggestions for improvement, please contact us:

  • Email: pafdcpm@cityofpa.us

We try to respond to feedback within 2 business days.
